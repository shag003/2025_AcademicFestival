# 2025_AcademicFestival
### **1. YOLOv8로 사람 객체 검출**

- `ultralytics`의 YOLOv8 모델을 사용해 이미지에서 'person' 객체를 자동으로 탐색.
- 객체가 검출되면 해당 영역만 crop해서 분석하고, 검출되지 않으면 전체 이미지를 분석.

### **2. 핵심 시각 특징 추출**

- crop된 영역(또는 전체 이미지)에서 다음과 같은 시각적 특징을 계산:
    - **밝기(brightness)**: 전체 픽셀의 평균값
    - **색상 다양성(color_std)**: 픽셀 값의 표준편차로 색상 변화 정도
    - **채도(saturation)**: RGB의 최대-최소값 차이의 평균
    - **복잡도(complexity)**: Laplacian 필터를 통한 텍스처 복잡도

### **3. 특징값에 노이즈 추가**

- 특징값에 작은 가우시안 노이즈를 더해반복 평가 시 결과에 미세한 변화(무작위성)가 나타나도록 함.

### **4. 그룹별 선호도 산출**

- **발달 그룹:** 밝기와 화려함(색상 다양성, 채도, 복잡도)에 모두 민감하게 반응하도록 가중치(예: `[1.3, 2.0, 2.0, 2.2]`)를 부여
- **비발달 그룹:** 밝기에 크게 반응하고, 나머지 특징에는 약하게 반응하도록 가중치(예: `[2.5, 0.3, 0.3, 0.3]`)를 부여
- 각 그룹에 대해 특징값과 가중치를 곱해 선호도를 계산하고, 1~5점으로 변환

### **5. 동기값 산출**

- 선호도 점수를 sigmoid 함수로 변환하여 동기값을 산출(선호도가 높을수록 동기값도 높아짐)

### **6. 별점 시각화**

- 점수를 반올림하여 1~5개의 별(★)로 시각적으로 표시결과를 한눈에 보기 쉽게 제공.

### **7. GUI 환경**

- 사용자가 이미지를 선택하면,이미지와 결과를 GUI 창에 즉시 출력.
